## 修正內容
1.  **繪圖座標修正**：修正了 Canvas 在縮放 (Scale) 後，清空畫布範圍過大的問題。原本畫布被黑色覆蓋，現在已修正為正確的網格座標。
2.  **初始化邏輯優化**：確保在 DOM 載入後執行初始化，並正確設定繪圖比例。
3.  **操作優化**：增加了按鍵後的立即重畫邏輯，並支援 Hard Drop 與雙向旋轉。

## 檔案結構
- [index.html](./index.html)：結構與 UI。
- [style.css](./style.css)：視覺設計與霓虹特效。
- [game.js](./game.js)：遊戲核心邏輯。
- [audio.js](./audio.js)：Web Audio API 音效。

## 操作說明
- `← / →`：移動
- `↑ / X`：順時針旋轉
- `Z`：逆時針旋轉
- `↓`：加速下落 (Soft Drop)
- `Space`：直接下落 (Hard Drop)
